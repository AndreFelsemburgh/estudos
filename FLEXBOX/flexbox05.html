Propriedade justify-content

Propriedade de alinhamento dos elementos, uma propriedade que vai controlar diretamente os flex-items.

Vai definir o alinhamento dos flex-items de acordo com o o eixo main-axis, ou seja, de acordo com o eixo principal

Essa propriedade ela está diretamente ligada com o flex-direction. Se configurarmos o flex-direction: row; os flex-items serão alinhados da esquerda para direita. logo o justify-content: flex-start; estará da esquerda para direita. Porém, se configurarmos o flex-direction: row-reverse; os flex-items serão alinhados da direita para a esquerda, logo o justify-content: flex-start; será da direita para esquerda.

Veja os exemplos abaixo:

flex-direction: row;
justify-content: flex-start

Para o justify-content: space-between; os elementos flex-items serão distribuídos com espaços iguais entre eles. O primeiro elemento ficará posicionado no flex-start e o último elemento será posicionado no flex-end. Os demais serão distribuídos com espaçamento iguais entre esses dois elementos.
