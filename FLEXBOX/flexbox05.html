Propriedade justify-content

Propriedade de alinhamento dos elementos, uma propriedade que vai controlar diretamente os flex-items.

Vai definir o alinhamento dos flex-items de acordo com o eixo principal

Como essa propriedade ela está diretamente ligada com o flex-direction, se configurarmos o flex-direction: row;, o eixo principal será o main-axis e, por padrão, os flex-items serão alinhados da esquerda para direita.

Bem como, se configurarmos o flex-direction: column;, o eixo principal será o cross-axis e, por padrão, os flex-items serão alinhados de cima para baixo

Veja os exemplos abaixo:

flex-direction: row;
justify-content: flex-start

Para o justify-content: space-between; os elementos flex-items serão distribuídos com espaços iguais entre eles. O primeiro elemento ficará posicionado no flex-start e o último elemento será posicionado no flex-end. Os demais serão distribuídos com espaçamento iguais entre esses dois elementos.

justify-content: space-between
